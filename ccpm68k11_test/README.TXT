CompuPro CP/M-68K v1.1I Test Package
====================================

This is a test package for SIMH/AltairZ80 that contains a blank 20MB hard
disk image and a floppy disk image with some utilities on it.

Instead of this package, use the CompuPro CP/M-68K v1.1I package located
here:
https://github.com/hharte/altairz80-packages/tree/main/ccpm68k11
