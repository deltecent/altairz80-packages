CompuPro CP/M-68K v1.1I
=======================

In November, 2022, the CompuPro CP/M-68K v1.1I System Master disks were
recovered from orignal 8" floppies and archived at:
https://bitsavers.org/bits/Compupro/ 

CompuPro CP/M-68K v1.1I is distributed per the license here:
http://www.cpm.z80.de/license.html

This distribution was created using the SIMH/AltairZ80 simulator
(https://github.com/open-simh/simh) from the System Master disks
and supports the following CompuPro components:

CompuPro 68000 CPU Board
CompuPro System Support Board
CompuPro DISK1A floppy disk controller
CompuPro DISK2 hard disk controller
128K of RAM

A SIMH configuration file is provided:

ccpm68k11.ini

The disk image files can be written to floppies using Dave Dunfield's
ImageDisk software.

CP/M-68K can be altered to support more than 128K of RAM by following
the instructions here:
https://bitsavers.org/pdf/compupro/CPM_68K_Technical_Manual_and_Installation_Feb84.pdf

Download link: https://github.com/hharte/altairz80-packages/tree/main/ccpm68k11
